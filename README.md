# GreenRibbon Schleifendruck-Plattform

Die GreenRibbon Kranzschleifendruck-Plattform bietet eine einzigartige Lösung zur web-basierten Gestaltung und Bestellabwicklung von Kranz- oder Schmuckschleifen.

Die Plattform ist die ideale Ergänzung zur Schleifendrucksoftware [GreenRibbon](https://www.greenribbon.de) Desktop. Sie erlaubt die Erstellung von Kranzschleifen  durch den Kunden direkt im Webbrowser (Firefox, Chrome, Safari, Edge) und deckt den vollständigen Prozess der Bestellabwicklung beim Betreiber der Plattform ab.

## Entwicklung

Für die Entwicklung und zum kompilieren wird eine [XOJO](https://www.xojo.com) Professional Lizenz benötigt.
Im Verzeichnis "Source" befinden sich das Projekt und alle erfordelriche Quelldateien.

## Betrieb

Die kompilierte Software kann unter Linux betrieben werden. MacOS oder Windows sind prinzipiell ebenfalls möglich, jedoch ist das Hosting mit Linux häufig kostengünstiger. Der Hostingserver sollte über mindestens 16GB RAM un 4 CPUs verfügen. Es wird keine Datenbank benötigt.
